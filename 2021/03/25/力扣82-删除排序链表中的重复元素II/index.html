82. 删除排序链表中的重复元素II

https://leetcode-cn.com/problems/remove-duplicates-from-sorted-list-ii/

存在一个按升序排列的链表，给你这个链表的头节点 head ，请你删除链表中所有存在数字重复情况的节点，只保留原始链表中 没有重复出现 的数字。

返回同样按升序排列的结果链表。

示例 1：

输入：head = [1,2,3,3,4,4,5]
输出：[1,2,5]

示例 2：

输入：head = [1,1,1,2,3]
输出：[2,3]

提示：

链表中节点数目在范围 [0, 300] 内
-100 <= Node.val <= 100
题目数据保证链表已经按升序排列

 学到的知识点：emmm不知道有没有
 什么？链表插入删除相关的题目不就一种方法嘛？
 先创建一个relPre节点用于指向head节点（基本操作）
 对于本题，设置一个遍历整个列表一次的指针pointer，每次比较pointer.val是否等于pointer.next.val，如果等于就一直往后找到一个不等元素，再把前面和后面连接起来。

最终代码如下：

public class Solution {
    public ListNode DeleteDuplicates(ListNode head) {
        ListNode realPre = new ListNode(int.MinValue,head);
        ListNode pointer = realPre;
        ListNode pre = realPre;
        while(pointer!=null && pointer.next!=null){
            if(pointer.val==pointer.next.val){
                while(pointer.next!=null&&pointer.val==pointer.next.val){
                    pointer=pointer.next;
                }
                pre.next=pointer.next;
                pointer=pre;
            }
            pre=pointer;
            pointer=pointer.next;
        }
        return realPre.next;
    }
}
